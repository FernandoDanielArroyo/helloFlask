`Installation et utilisation de Poetry`

```
syntaxe markdown = https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax#quoting-code

installer poetry = curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python -

voir la version = poetry --version

exemple de création poetry = 
$ cd ~/Documents/
$ poetry new cow-counts
$ cd cow-counts
$ touch cow_counts/mycow.py
$ poetry add cowsay

installer et utiliser appli avec poetry run = 
$ poetry install
$ poetry run count Hello world 

Pour partager poetry =
$ poetry build 

initialiser un nouveau porjet poetry = 
$ poetry init --no-interaction

ajouter une dépendance :
$ poetry add pandas

```
